# 🏬 Warehouse Management System (Python OOP Project)

This project is a **mini warehouse + order + shipment management system** built with Python using:

- Object Oriented Programming (OOP)
- Dataclasses
- UUID identifiers
- Composition & basic system design
- CLI (menu-based interface)

---

# 🚀 Features

## 📦 Warehouse Management
- Add warehouse
- Remove warehouse
- Store products inside warehouses

## 📦 Product Management
- Add products to warehouse
- Remove products from warehouse

## 🧾 Order System
- Create orders
- Link orders to customers
- Track order history

## 🚚 Shipment System
- Create shipment from order
- Track shipment status:
  - preparing
  - shipped
  - delivered

## 📊 System Management
- Central system controller (`WarehouseSystem`)
- Transaction history logging
- Full system tracking

---

# 🧠 Concepts Used

- Python OOP
- Inheritance
- Composition
- Dataclasses
- Property decorators
- Encapsulation (private fields)
- UUID unique IDs
- CLI menu system

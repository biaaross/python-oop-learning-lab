# 🚗 Vehicle Rental System (Python OOP Project)

## 📌 Project Overview

This is a simple **Object-Oriented Programming (OOP) based Vehicle Rental System** built with Python.

The project simulates a real-world car rental workflow including:

- Adding vehicles to a fleet
- Registering customers
- Renting vehicles
- Returning vehicles
- Tracking customer rental history

It also includes a **Command Line Interface (CLI)** for user interaction.

---

## 🎯 Purpose of the Project

The main goal of this project is to practice and demonstrate:

- Object-Oriented Programming (OOP)
- Class relationships (composition)
- Encapsulation
- Real-world system design thinking
- CLI application development

---

## 🧠 Features

### 🚗 Car Management
- Add new cars to the system
- Track availability status (rented / available)
- Store vehicle information (brand, model, year, price)

### 👤 Customer Management
- Register new customers
- Store rental history per customer
- Track active rentals

### 🔄 Rental System
- Rent a car to a customer
- Return rented cars
- Prevent double booking

### 📊 System Features
- View all cars
- View all customers
- Basic transaction tracking
